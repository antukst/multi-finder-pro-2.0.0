# Multi-Finder Pro V2 ??
A powerful, universal browser extension for multi-word highlighting with a sleek, animated Lofi-style UI.

## ? Features
- **Floating Panel**: A draggable and resizable in-page search window.
- **Advanced Engine**: Support for Case Sensitivity, Whole Words, and Diacritics.
- **Trap Detector**: Specialized, independent highlighting for "Survey Trap Questions".
- **Glow Highlights**: Optional glow effects for better visibility.
- **Context Menu**: Right-click any text to "Add to Multi Find".
- **Shortcuts**: Use `Ctrl+Shift+Y` to toggle the in-page panel.
- **Persistent Highlighting**: Highlighting remains active after page reloads or navigation.
- **Bulk Import/Export**: Easy management of large keyword lists.

## ?? Installation
1. Download or Clone the project.
2. Go to `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the folder.

## ?? Privacy Policy (Chrome Web Store)
The Chrome Web Store requires a direct link to a valid privacy policy page (not a generic homepage). A simple option is to host the `privacy-policy.html` file included in this repository using GitHub Pages, Netlify, or any static hosting provider.

When submitting/updating your item in the Chrome Web Store, set the Privacy Policy URL to a page that directly shows the policy (e.g., `https://<your-domain>/privacy-policy.html`).

## ?? Shortcuts
- **Open Popup**: `Ctrl+Shift+F` (Default)
- **Toggle In-Page UI**: `Ctrl+Shift+Y`
